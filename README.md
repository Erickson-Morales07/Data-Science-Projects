# Data Science Projects

## Notebooks
'/cat vs dog/' - A computer vision system that distinguishes between cats and dogs using convolutional neural networks (CNNs).
'/Credict Card Fraud Detection/' - Machine learning algorithm to identify fraudulent transactions. It employs data imbalance techniques.
'/movie system recomendation/' - Movie recommendation engine that suggests content based on similarity of characteristics.
'/Multiple Linear Regression/' - Multiple linear regression model for predicting outcomes based on multiple independent variables. Ideal for predictive analytics.
'/sentiment analysis/' - Text classifier that determines the emotional polarity (positive/negative) in comments or reviews. Applicable to social media monitoring and customer feedback analysis.
